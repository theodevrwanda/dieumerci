# NDM Portfolio - Niyonsenga DieuMerci

Professional portfolio application for **Niyonsenga DieuMerci**, a versatile IT Professional, Web Designer, and Software Developer based in Rwanda.

## 🚀 About This App

This application is a modern, premium portfolio built to showcase technical expertise in:
- **Web Design:** Responsive and dynamic interfaces using React and Tailwind CSS.
- **Software Development:** Innovative solutions tailored to community needs.
- **Network Configuration:** Professional infrastructure and connectivity setups.
- **IT Support:** Comprehensive technical know-how and system installations.

The design is inspired by **Apple's aesthetic**—clean, minimalist, and high-performance, featuring:
- **Glassmorphism:** Elegant blurred backgrounds and translucent surfaces.
- **Animations:** Smooth, high-frame-rate transitions powered by Framer Motion.
- **Interactive UI:** Dynamic "Border Beam" animations and premium card hover effects.
- **AI Integration:** Instagram-style chat interface linked to the Gemini Pro API.

## 💻 Local Development

1. **Clone the repository:**
   ```sh
   git clone <repository-url>
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Start the development server:**
   ```sh
   npm run dev
   ```

4. **Environment Variables:**
   Ensure you have a `.env` file with the required API keys for Gemini and EmailJS.
